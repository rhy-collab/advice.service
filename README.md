# Charter Law

Working files for **Charter Law** — an AI-native, attorney-reviewed contract-review service for growth-stage startups, modelled on General Legal (YC W26) and built solo (founder vibe-coding) with one per-matter reviewing attorney.

## The documents

**Build the software from `charter-law-super-prompt.md`** — it's the authoritative feature spec and points to the rest. Read order for building: super-prompt → roadmap → tech-stack → operating-playbook. Read order for go-to-market: ad-library → video-scripts → growth-and-channels. Research backing it all: dossier + Competitors.

| File | What it is |
|------|------------|
| **`charter-law-super-prompt.md`** | ⭐ The single authoritative build brief for the software: full feature map (incl. the playbook system, attorney workbench, feedback loop, status tracker, "Clerk" assistant), production-readiness rules, build sequencing, and the copy-paste Claude Code prompt. Drive the build from this, one phase at a time. |
| **`charter-law-roadmap.md`** | The phased build plan (Phase 0 legal foundation → manual MVP → marketing site → portal → AI engine + attorney app → scale). Owners, dependencies, "done =" definitions, costs. |
| **`charter-law-tech-stack.md`** | The decided technology stack (Vite/React + FastAPI + Postgres + Clerk + Stripe + Cloud Run + Claude), matching General Legal. Plain-English explanations, project structure + data model. |
| **`charter-law-operating-playbook.md`** | How to *run* the business: the YC AI-native-services framework + General Legal's operating model → pricing, economics (COGS/margin), variance, the early-demand trap, the corporate/compliance structure, growth motion. |
| **`charter-law-ad-library.md`** | 71 Charter Law ad concepts + a full A/B teardown of General Legal's 71 ads (format, real text, offer, what they tested). Grouped by claim family, in Charter Law's voice, with truthfulness/compliance flags. |
| **`charter-law-video-scripts.md`** | Designer-ready 10–15s scripts for every video-flagged ad concept — shot-by-shot, on-screen text, end cards, LinkedIn specs. |
| **`charter-law-growth-and-channels.md`** | Why LinkedIn ads is the primary channel, the channel funnel (LinkedIn/Google/Meta), CPL economics, and how this business actually wins (distribution + trust + compounding moat). For a solo bootstrap. |
| **`general-legal-dossier.md`** | Deep research on the anchor competitor: company, founders, ~11-attorney roster, funding, traction, full pricing, the Sentinel workflow, complete tech-stack fingerprint, MCP spec, the 71-ad marketing teardown, corporate structure, and strategic lessons. |
| **`Competitors.md`** | 19-firm AI-native-law-firm dossier (Crosby, Moritz, Manifest, Covenant, Tilder/Tacit, LegalOS, Vector, Avantia + more): stacks, models, architecture, pricing, security — the source of the super-prompt's `[NEW]` features. |
| **`architecture.html`** | Visual architecture diagram of the system. |
| **`digital-twins/`** | Source-grounded founder personas (General Legal's founders) for voice/strategy study. |
