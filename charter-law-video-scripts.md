# Charter Law — Video Ad Scripts (designer-ready)

> 10–15 second scripts for every video-flagged concept in `charter-law-ad-library.md`. Each is shot-by-shot with on-screen text, timing, and an end card, so a motion designer can build straight from it.
> All are **LinkedIn-feed first.** Pair with the ad library for the matching single-image variants.

---

## Global specs (apply to every script)

- **Length:** 10–15s. **Hook in the first 1.5s** (LinkedIn autoplays muted in-feed — the opening text frame must land the whole idea with sound off).
- **Aspect ratios:** deliver **1:1 (1080×1080)** and **4:5 (1080×1350)** for feed; **9:16 (1080×1920)** optional for Stories/mobile.
- **Sound-off by default:** all dialogue/VO must be **burned-in captions**; design works silently, music is a bonus not a dependency.
- **Style:** clean kinetic typography + simple motion graphics (fast, cheap to produce, on-brand for "legal that runs like software"). Live-action optional where noted.
- **Brand:** Charter Law wordmark; site `charterlaw.services`; restrained palette (one accent colour); a clear, legible sans-serif.
- **End card (last 2–2.5s, same on all):** logo + one-line value prop + CTA button look. Default CTA: **"Start a review → charterlaw.services"** (swap to "First review free" once that offer is live).
- **Compliance:** respect the flags. **[P0]** scripts assert attorney involvement → only run once the reviewing-attorney structure is live and wording is attorney-approved; until then use the interim line noted. **[SLA]** → only state a turnaround you can actually hit. Don't imply AI gives the advice — AI prepares, the attorney decides.

## Production notes for the designer

- **Build order (cheapest → priciest):** Script 5 → 2 → 4 → 3 → 7 → 6 → 1. Start with the pure-typography ones to test the hook cheaply; only invest in live-action (Script 1) once a hook is proven.
- **Make a reusable end-card template** (logo + value prop + CTA) and a **caption style** once; reuse across all seven for a consistent system and faster turnaround.
- **Deliver each as:** 1:1 + 4:5 (and 9:16 if budget allows), captions burned in, MP4 ≤ ~30MB, plus a muted-friendly thumbnail frame.
- **Versioning for A/B:** keep the visuals fixed and **swap only the opening hook text frame** to test hooks within a concept (mirrors how General Legal retested the same story with different openers).
- **Compliance gate:** Scripts 6 & 7 are **[P0]** — don't publish until the reviewing-attorney structure is live and the lines are attorney-approved. Script 1 & 2 are **[SLA]** — only state "hours / 3 hours" if you can hit it.
